标题

000035993 - Versal：Vivado 2023.2 - Advanced IO Wizard - 启用调试端口无效

描述

在 Advanced IO Wizard 的“Advanced”（高级）选项卡中，包含“Enable Debug Ports”（启用调试端口）选项，PG320 中对该选项的描述如下：

启用该选项时，将显示 ILA 相关的所有调试端口。选中该选项时，将显示“debug_bus”端口。该调试总线可用于通过 ILA 执行调试

在 Advanced IO Wizard 中，当“Debug_bus”实例接地时，存在一个已知问题。 

解决方案

要解决此问题，您需要手动连接到 Advanced IO Wizard 中的信号线。 

例如，在“Advanced IO Wizard”中自定义 IP，在“Basic”（基本）选项卡中，将应用配置设为“Source Synchronous”（源同步），将总线方向设为“Bidir+ Tx +Rx”（双向 + 发射 + 接收），并选中“Single strobe”（单选通）模式，在“Advanced”选项卡中勾选“Enable Debug Ports”（启用调试端口）选项。生成 OOC 并对设计进行综合。 



打开已综合的设计时，会出现“Setup Debug Core”（设置调试核）选项，可用于手动插入 ILA 并从已综合的网表添加所有调试端口。
